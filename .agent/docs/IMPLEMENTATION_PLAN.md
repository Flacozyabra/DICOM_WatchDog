# План реализации: Совместимость с PyQt5 и автоматическая сборка релизов v1.1.0

Этот план описывает шаги по интеграции шима совместимости с PyQt5 (для запуска без MSVC++) и созданию скриптов автоматической сборки исполняемых файлов для PyQt6 и PyQt5.

## User Review Required

> [!IMPORTANT]
> Для сборки версии PyQt5 на сборочном компьютере в виртуальном окружении (`venv`) должна быть установлена библиотека `PyQt5`. Скрипт сборки попытается установить её автоматически через `pip`, если она отсутствует.

## Proposed Changes

### 1. Версионирование и совместимость

#### [MODIFY] [config_utils.py](file:///c:/Users/Falco/Desktop/DICOM%20WatchDog/core/config_utils.py)
* Обновить константу версии до `VERSION = "1.1.0"`.

#### [MODIFY] [main.py](file:///c:/Users/Falco/Desktop/DICOM%20WatchDog/main.py)
* Внедрить шим-совместимости PyQt6 -> PyQt5 на самом старте приложения. 
* При импорте PyQt6, если происходит сбой (или установлена переменная окружения `FORCE_PYQT5=1`), шим будет:
  1. Загружать `PyQt5`.
  2. Патчить классы `PyQt5` (добавляя недостающие вложенные перечисления, которые используются в синтаксисе PyQt6).
  3. Подменять `sys.modules['PyQt6']` ссылками на модули `PyQt5`.
* Если не удалось импортировать ни PyQt6, ни PyQt5, показывать нативное окно ошибки с требованием установить MSVC++.

### 2. Скрипты сборки релизов

#### [NEW] [build_PyQt6.py](file:///c:/Users/Falco/Desktop/DICOM%20WatchDog/build_PyQt6.py)
* Скрипт для сборки версии приложения на PyQt6.
* Автоматически генерирует `src/app_icon.ico` из `src/logo.png` при помощи Pillow.
* Запускает PyInstaller со следующими параметрами:
  - Имя: `DICOM_WatchDog_v1.1.0_PyQt6`
  - Режим: один файл (`--onefile`), без консоли (`--windowed`)
  - Ресурсы: добавление папок `src` (`--add-data="src;src"`) и `themes` (`--add-data="themes;themes"`)
  - Скрытые импорты: автоматический сбор подмодулей `pydicom` и `pynetdicom`
  - Исключения: удаление `PyQt5` из сборки (`--exclude-module=PyQt5` и др.)

#### [NEW] [build_PyQt5.py](file:///c:/Users/Falco/Desktop/DICOM%20WatchDog/build_PyQt5.py)
* Скрипт для сборки версии приложения на PyQt5.
* Аналогичен первому скрипту, но:
  - Устанавливает переменную окружения `FORCE_PYQT5 = "1"` перед запуском сборки.
  - Имя файла: `DICOM_WatchDog_v1.1.0_PyQt5`
  - Исключения: удаление `PyQt6` из сборки (`--exclude-module=PyQt6` и др.)
  - Автоматически проверяет и устанавливает `PyQt5` в `venv` перед компиляцией.
